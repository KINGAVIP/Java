Jsp and servlet

JSP- stand for Java Server Pages
(pages like login.jsp are formed in java which are render through servlet)

-is a server side technology
-used to create dynamic web content
-JSP tags are used to convert java code into Html pages
-has access to entire api of java
-extended version of servlet

servlet

-work on the server side
-capable of handling complex requests obtained from the webserver
and able to process it and then able to provide the dynamic response accordingly

-are platform independent
-JDBC extends servlet class for connectivity
